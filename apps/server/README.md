# Built On Gno APIs Service

## Description

[Express](https://github.com/expressjs/express) framework TypeScript starter repository.

## Installation

First, copy env file:

```bash
cp env.example .env
```

Edit environment variable to your configuration

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run dev

# build for production
$ pnpm run build
```
